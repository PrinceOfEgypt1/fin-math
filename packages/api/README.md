# @finmath/api

API REST para cálculos de matemática financeira.

## Quick Start

```bash
pnpm install
pnpm dev           # Development
pnpm test          # Tests
pnpm build         # Build
```

## Endpoints

- `GET /health` - Health check
- `GET /api-docs` - Swagger UI

## Stack

- Fastify 4.26
- Pino (logging)
- Zod (validation)
- Vitest (tests)
